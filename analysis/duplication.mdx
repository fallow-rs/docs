title: "Code Duplication"
description: "Find copy-pasted code blocks across your codebase."
---

`fallow dupes` finds duplicated code blocks across your entire codebase. 4-75x faster than jscpd on real-world projects.

```bash
fallow dupes
```

## Detection modes

| Mode | What it catches |
|:-----|:----------------|
| **strict** | Exact token-for-token clones |
| **mild** | Default — normalizes syntax variations (whitespace, semicolons) |
| **weak** | Clones with different string literal values |
| **semantic** | Clones with renamed variables and different literals |

```bash
fallow dupes --mode strict
fallow dupes --mode semantic    # Catch clones with renamed variables
```

## Thresholds and limits

```bash
fallow dupes --min-tokens 50   # Minimum tokens per clone (default: 50)
fallow dupes --min-lines 5     # Minimum lines per clone (default: 5)
fallow dupes --threshold 5     # Fail if duplication exceeds 5%
fallow dupes --skip-local      # Only cross-directory duplicates
```

## Clone families

Clone groups sharing the same file set are grouped into **clone families** with refactoring suggestions:

- **Extract function** — when clones are in the same file
- **Extract module** — when clones span multiple files

## Cross-language detection

Compare TypeScript and JavaScript files by stripping type annotations:

```bash
fallow dupes --cross-language
```

This normalizes `.ts` files to their `.js` equivalent for comparison, catching clones where one was converted from TypeScript to JavaScript or vice versa.

## Baseline comparison

Adopt duplication limits incrementally:

```bash
# Save current duplication as baseline
fallow dupes --save-baseline

# Fail only on new duplication
fallow dupes --baseline
```

## Debugging

Trace all clones of a specific code location:

```bash
fallow dupes --trace src/utils.ts:42
```

## Benchmarks vs jscpd

| Project | Files | fallow | jscpd | Speedup |
|:--------|------:|-------:|------:|--------:|
| [zod](https://github.com/colinhacks/zod) | 174 | **45ms** | 905ms | **20.1x** |
| [fastify](https://github.com/fastify/fastify) | 286 | **546ms** | 2.00s | **3.7x** |
| [preact](https://github.com/preactjs/preact) | 244 | **18ms** | 1.32s | **73.8x** |

Fallow uses a suffix array with LCP for clone detection — no quadratic pairwise comparison.
