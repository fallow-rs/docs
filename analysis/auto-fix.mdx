---
title: "Auto-fix"
description: "Automatically remove unused exports and dependencies."
---

`fallow fix` removes unused exports and unused dependencies from your codebase.

## Preview changes

Always preview before applying:

```bash
fallow fix --dry-run
```

This shows exactly what would be removed without making any changes.

## Apply fixes

```bash
fallow fix
```

In interactive terminals, fallow asks for confirmation before applying. In CI or non-TTY environments, use `--yes` or `--force`:

```bash
fallow fix --yes
```

## What gets fixed

- **Unused exports** — the `export` keyword is removed, keeping the declaration
- **Unused dependencies** — removed from `package.json`

<Warning>
Auto-fix does not delete entire files or remove unused enum/class members. Use `fallow check` to identify those and remove them manually.
</Warning>

## JSON output

For scripting and CI:

```bash
fallow fix --dry-run --format json
```
