<Tabs>
  <Tab title="npm (recommended)">
    Prebuilt binaries for macOS, Linux, and Windows — no Rust toolchain required.

    <Snippet file="install.mdx" />

    <Tip>
      The npm package uses `optionalDependencies` to download the correct platform binary automatically.
    </Tip>
  </Tab>

  <Tab title="Cargo">
    Install from crates.io if you have a Rust toolchain:

    ```bash
    cargo install fallow-cli
    ```
  </Tab>

  <Tab title="Binary download">
    Download prebuilt binaries from [GitHub Releases](https://github.com/fallow-rs/fallow/releases).

    | Target | Platform |
    |:-------|:---------|
    | `fallow-darwin-arm64` | macOS Apple Silicon |
    | `fallow-darwin-x64` | macOS Intel |
    | `fallow-linux-x64-gnu` | Linux x64 (glibc) |
    | `fallow-linux-arm64-gnu` | Linux ARM64 (glibc) |
    | `fallow-linux-x64-musl` | Linux x64 (musl) |
    | `fallow-linux-arm64-musl` | Linux ARM64 (musl) |
    | `fallow-win32-x64-msvc.exe` | Windows x64 |
  </Tab>
</Tabs>

## Verify installation

```bash
fallow --version
```

<Check>
  If you see a version number, you're all set. Head to the [Quick Start](/quickstart) to run your first analysis.
</Check>
