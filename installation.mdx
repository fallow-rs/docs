---

## npm (recommended)

Prebuilt binaries for macOS, Linux, and Windows — no Rust toolchain required.

```bash
# Run directly
npx fallow check

# Install globally
npm install -g fallow
```

The npm package uses `optionalDependencies` to download the correct platform binary automatically (darwin-arm64, darwin-x64, linux-x64-gnu, linux-arm64-gnu, linux-x64-musl, linux-arm64-musl, win32-x64-msvc).

## Cargo

Install from crates.io if you have a Rust toolchain:

```bash
cargo install fallow-cli
```

## GitHub Releases

Download prebuilt binaries directly from [GitHub Releases](https://github.com/fallow-rs/fallow/releases).

Available targets:
- `fallow-darwin-arm64` — macOS Apple Silicon
- `fallow-darwin-x64` — macOS Intel
- `fallow-linux-x64-gnu` — Linux x64 (glibc)
- `fallow-linux-arm64-gnu` — Linux ARM64 (glibc)
- `fallow-linux-x64-musl` — Linux x64 (musl)
- `fallow-linux-arm64-musl` — Linux ARM64 (musl)
- `fallow-win32-x64-msvc.exe` — Windows x64

## Verify installation

```bash
fallow --version
```
