## Run fallow

No configuration needed. From your project root:

```bash
npx fallow check
```

Fallow auto-detects your frameworks via `package.json` and reports unused files, exports, dependencies, types, and more.

## Find duplicated code

```bash
npx fallow dupes
```

This finds copy-pasted code blocks across your entire codebase using the default `mild` detection mode.

## Preview auto-fixes

Before making changes, preview what fallow would remove:

```bash
npx fallow fix --dry-run
```

When you're ready, apply the fixes:

```bash
npx fallow fix
```

## Create a config file

For customization, generate a config file:

```bash
fallow init
```

This creates `.fallowrc.json` with a `$schema` field for IDE autocomplete. Use `fallow init --toml` for TOML format.

```jsonc
// .fallowrc.json
{
  "$schema": "https://raw.githubusercontent.com/fallow-rs/fallow/main/schema.json",
  "entry": ["src/workers/*.ts", "scripts/*.ts"],
  "ignorePatterns": ["**/*.generated.ts"],
  "rules": {
    "unused-files": "error",
    "unused-exports": "warn",
    "unused-types": "off"
  }
}
```

## Migrating from knip or jscpd?

Fallow can auto-migrate your existing config:

```bash
fallow migrate            # Auto-detect and migrate
fallow migrate --dry-run  # Preview without writing
```
