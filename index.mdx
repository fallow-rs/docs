## Why fallow?

Dead code and duplication increase bundle sizes, slow CI, and make codebases harder to navigate. AI-assisted development accelerates this — agents generate code but rarely suggest deletions.

<CardGroup cols={3}>
  <Card title="Zero config" icon="bolt">
    Works out of the box. 47 built-in plugins auto-detect Next.js, Vite, Jest, Tailwind, and 40+ more.
  </Card>
  <Card title="Sub-second" icon="gauge-high">

  </Card>

## Speed comparison

<Tabs>
  <Tab title="vs knip (dead code)">
    | Project | Files | fallow | knip | Speedup |
    |:--------|------:|-------:|-----:|--------:|
    | [zod](https://github.com/colinhacks/zod) | 174 | **16ms** | 582ms | **35.8x** |
    | [fastify](https://github.com/fastify/fastify) | 286 | **20ms** | 804ms | **39.4x** |
    | [preact](https://github.com/preactjs/preact) | 244 | **29ms** | 771ms | **26.6x** |
    | synthetic (1,000 files) | 1,000 | **45ms** | 378ms | **8.4x** |
    | synthetic (5,000 files) | 5,000 | **200ms** | — | — |

    <Info>
      Speedup narrows on larger projects as analysis time dominates over startup: 25-40x on small projects, 8-10x on 1,000+ file projects. Fallow stays sub-second even at 5,000 files.
    </Info>
  </Tab>
  <Tab title="vs jscpd (duplication)">
    | Project | Files | fallow | jscpd | Speedup |
    |:--------|------:|-------:|------:|--------:|
    | [preact](https://github.com/preactjs/preact) | 244 | **18ms** | 1.32s | **73.8x** |
    | [zod](https://github.com/colinhacks/zod) | 174 | **45ms** | 905ms | **20.1x** |
    | [fastify](https://github.com/fastify/fastify) | 286 | **546ms** | 2.00s | **3.7x** |

    <Info>
      Fallow uses a suffix array with LCP for clone detection — no quadratic pairwise comparison.
    </Info>
  </Tab>
</Tabs>

## What it finds

<CardGroup cols={2}>
  <Card title="Dead code" icon="skull-crossbones">
    10 issue types: unused files, exports, types, dependencies, devDependencies, enum members, class members, unresolved imports, unlisted deps, duplicate exports.
  </Card>
  <Card title="Code duplication" icon="clone">
    4 detection modes from exact matches to semantic clones with renamed variables. Clone families with refactoring suggestions.
  </Card>
  <Card title="Auto-fix" icon="wand-magic-sparkles">
    Remove unused exports and dependencies automatically. Preview with `--dry-run` before applying.
  </Card>
  <Card title="CI-ready" icon="circle-check">
    SARIF output for GitHub Code Scanning, baseline comparison for incremental adoption, `--changed-since` for PR-only checks.
  </Card>
</CardGroup>
