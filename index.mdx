Fallow finds unused files, exports, dependencies, and types — plus duplicated code blocks across your entire codebase. 3-40x faster than [knip](https://knip.dev) for dead code, 4-75x faster than [jscpd](https://github.com/kucherenko/jscpd) for duplication, with no Node.js runtime dependency.

```bash
npx fallow check    # Dead code analysis
npx fallow dupes    # Duplication detection
```

## Why fallow?

Dead code and duplication increase bundle sizes, slow CI, and make codebases harder to navigate. AI-assisted development accelerates this — agents generate code but rarely suggest deletions.

Fallow combines dead code analysis and duplication detection in a single sub-second tool: one install, one config, one CI step.
