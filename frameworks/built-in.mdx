## AST-based config parsing

The top 10 plugins have deep config parsing via Oxc — they read your config files (`vite.config.ts`, `jest.config.js`, etc.) and extract entry points, dependencies, and other settings automatically:

ESLint, Vite, Jest, Storybook, Tailwind, Webpack, TypeScript, Babel, Rollup, PostCSS, Nuxt.

## Checking active plugins

See which plugins are active in your project:

```bash
fallow list --plugins
```
