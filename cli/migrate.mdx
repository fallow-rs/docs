---
title: "fallow migrate"
description: "Migrate config from knip or jscpd."
---

Auto-migrate your existing knip and/or jscpd configuration to fallow.

```bash
fallow migrate
```

## Options

| Flag | Description |
|:-----|:------------|
| `--toml` | Output as `fallow.toml` instead of `.fallowrc.json` |
| `--dry-run` | Preview the generated config without writing |
| `--from <PATH>` | Path to the config file to migrate |

## What it detects

Fallow auto-detects:

- `knip.json`, `knip.jsonc`, `.knip.json`, `.knip.jsonc`
- `knip` field in `package.json`
- `.jscpd.json`
- `jscpd` field in `package.json`

If both knip and jscpd configs are found, both are migrated into a single fallow config.

## Examples

```bash
# Auto-detect and migrate
fallow migrate

# Preview without writing
fallow migrate --dry-run

# Output as TOML
fallow migrate --toml

# Migrate a specific file
fallow migrate --from config/knip.json
```

Warnings are shown for any config fields that can't be migrated.
