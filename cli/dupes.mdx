---
title: "fallow dupes"
description: "Find code duplication across your codebase."
---

Detect copy-pasted code blocks across your entire codebase.

```bash
fallow dupes
```

## Options

### Detection

| Flag | Description |
|:-----|:------------|
| `--mode <MODE>` | Detection mode: `strict`, `mild` (default), `weak`, `semantic` |
| `--min-tokens <N>` | Minimum tokens per clone (default: 50) |
| `--min-lines <N>` | Minimum lines per clone (default: 5) |
| `--skip-local` | Only report cross-directory duplicates |
| `--cross-language` | Strip TS types for TS ↔ JS matching |

### Output

| Flag | Description |
|:-----|:------------|
| `-f, --format <FORMAT>` | Output format: `human` (default), `json`, `sarif`, `compact` |
| `--quiet` | Suppress progress output |
| `--threshold <N>` | Fail if duplication exceeds N% |

### Incremental

| Flag | Description |
|:-----|:------------|
| `--baseline` | Compare against saved baseline |
| `--save-baseline` | Save current duplication as baseline |

### Debugging

| Flag | Description |
|:-----|:------------|
| `--trace <FILE:LINE>` | Show all clones of code at a specific location |
| `--performance` | Show pipeline timing breakdown |

## Detection modes

| Mode | Normalizes |
|:-----|:-----------|
| **strict** | Nothing — exact token match |
| **mild** | Whitespace, semicolons, trailing commas |
| **weak** | + string literal values |
| **semantic** | + variable names and numeric values |

## Examples

```bash
# Default mild mode
fallow dupes

# Catch renamed-variable clones
fallow dupes --mode semantic

# Only cross-directory duplicates
fallow dupes --skip-local

# Fail CI if duplication > 5%
fallow dupes --threshold 5

# TS ↔ JS comparison
fallow dupes --cross-language

# Trace clones at a specific line
fallow dupes --trace src/utils.ts:42
```
