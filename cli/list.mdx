title: "fallow list"
description: "Show discovered files, entry points, and plugins."
icon: "terminal"
---

Inspect what fallow discovered in your project. Use this command to debug configuration issues or verify that the right files, entry points, and plugins are being picked up.

<Tip>
  Use `fallow list --plugins` to verify which of the 79 built-in plugins are active for your project.
</Tip>

```bash
fallow list
```

## Options

| Flag | Description |
|:-----|:------------|
| `--files` | Show all discovered files |
| `--entry-points` | Show entry point files |
| `--plugins` | Show active plugins |
| `-f, --format <FORMAT>` | Output format: `human` (default), `json` |

## Examples

<CodeGroup>
```bash Inspect plugins
# Show active plugins
fallow list --plugins

# JSON output for scripting
fallow list --plugins --format json
```

```bash Inspect files
# Show all discovered files
fallow list --files

# Show entry points
fallow list --entry-points
```
</CodeGroup>
