---
title: "fallow init"
description: "Generate a config file."
---

Create a configuration file for your project.

```bash
fallow init
```

## Options

| Flag | Description |
|:-----|:------------|
| `--toml` | Generate `fallow.toml` instead of `.fallowrc.json` |

## Output

By default, creates `.fallowrc.json` with a `$schema` field for IDE autocomplete:

```jsonc
{
  "$schema": "https://raw.githubusercontent.com/fallow-rs/fallow/main/schema.json",
  "entry": [],
  "ignorePatterns": [],
  "rules": {}
}
```

With `--toml`, creates `fallow.toml`.
