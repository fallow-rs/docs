---
title: "fallow fix"
description: "Auto-remove unused exports and dependencies."
---

Automatically remove unused exports and dependencies.

```bash
fallow fix
```

## Options

| Flag | Description |
|:-----|:------------|
| `--dry-run` | Preview changes without applying |
| `--yes, --force` | Skip confirmation (required in non-TTY) |
| `-f, --format <FORMAT>` | Output format: `human` (default), `json` |

## Examples

```bash
# Preview what would be removed
fallow fix --dry-run

# Apply fixes (asks for confirmation)
fallow fix

# Apply without confirmation (CI)
fallow fix --yes

# JSON output for scripting
fallow fix --dry-run --format json
```

## What gets fixed

- **Unused exports** — the `export` keyword is removed, keeping the declaration
- **Unused dependencies** — removed from `package.json`

Unused files, enum members, and class members are not auto-fixed. Use `fallow check` to identify those.
